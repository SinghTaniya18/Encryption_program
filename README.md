# Encryption_program
🔐 Encryption Program (Python)

This is a simple encryption and decryption program built using Python. The project demonstrates basic concepts of cryptography by allowing users to encode and decode messages using a custom method.

Features:

* Encrypts plain text into coded text
* Decrypts encoded text back to original message
* Easy-to-use and beginner-friendly

Purpose:
This project was created to practice Python fundamentals and understand how basic encryption techniques work.

Future Improvements:

* Add stronger encryption algorithms
* Create a user interface
* Improve security and efficiency
